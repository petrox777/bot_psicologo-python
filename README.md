# 🤖 Bot Psicólogo em Python

## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de criar um chatbot capaz de conversar com o usuário de forma amigável e acolhedora, simulando uma conversa de apoio emocional.

O programa foi criado utilizando Python e funciona diretamente no terminal, identificando mensagens do usuário e respondendo de acordo com palavras-chave relacionadas a emoções, sentimentos e situações do dia a dia.

Além de ser um exercício de programação, o projeto também foi uma oportunidade para praticar lógica, organização de código e interação com o usuário.

---

## 🚀 Funcionalidades

- Conversa interativa pelo terminal;
- Identificação de sentimentos e emoções;
- Respostas personalizadas para diferentes situações;
- Sistema de saudação ao usuário;
- Fluxo de conversa contínuo;
- Tratamento de entradas de texto;
- Mensagens de apoio e incentivo.

---

## 🛠️ Tecnologias Utilizadas

- Python 3
- Estruturas condicionais (`if`, `elif`, `else`)
- Laços de repetição (`while`)
- Funções
- Manipulação de strings
- Entrada e saída de dados pelo terminal

---

## 🎯 Objetivos do Projeto

Este projeto foi desenvolvido para:

- Aprimorar conhecimentos em Python;
- Praticar lógica de programação;
- Aprender a estruturar projetos maiores;
- Trabalhar com análise de texto e interação com usuários;
- Desenvolver experiência prática para construção de portfólio.
